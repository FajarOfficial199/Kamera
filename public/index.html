Camera Controller Pro

Kontrol kamera dari jarak jauh dengan sistem room code. Host dapat mengontrol zoom, flash, dan mengambil screenshot dari client.

Buat Room (Host) Gabung Room (Client)

Cara Menggunakan

  • Host: Buat room baru dan bagikan kode room ke client
  • Client: Masukkan kode room yang diberikan host
  • Izin Kamera: Client harus memberikan izin akses kamera
  • Kontrol: Host dapat mengontrol zoom, flash, dan mengambil screenshot
  • Multi-Client: Satu room dapat memiliki beberapa client
