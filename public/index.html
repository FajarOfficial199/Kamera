Camera Controller

Kontrol kamera dari jarak jauh dengan sistem room code

Buat Room (Host) Gabung Room (Client)

Cara Penggunaan:

  • Host: Buat room baru, dapatkan kode, dan bagikan kode ke client
  • Client: Masukkan kode room yang diberikan oleh host
  • Host dapat mengontrol kamera client (zoom, senter, dll)
  • Client memberikan akses kamera untuk dapat dikontrol
